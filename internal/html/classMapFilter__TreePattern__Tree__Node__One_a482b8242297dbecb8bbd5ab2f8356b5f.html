MapFilter_TreePattern_Tree_Node_One::satisfy ( MapFilter_TreePattern_SatisfyParam $  param  ) 

Satisfy certain node type and let it's followers to get satisfied.

Satisfy node if there is one satisfied follower (any further followers mustn't be satisfied in order to pick up just first one of those). Mapping CAN'T continue after finding satisfied follower.
