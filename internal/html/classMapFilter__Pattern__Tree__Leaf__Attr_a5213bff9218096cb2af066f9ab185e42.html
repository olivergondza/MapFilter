MapFilter_Pattern_Tree_Leaf_Attr::satisfy ( MapFilter_Pattern_SatisfyParam $  param  ) 

Satisfy certain node type and let it's followers to get satisfied.

Satisfy node just if there are no unsatisfied follower. Finding unsatisfied follower may stop mapping since there is no way to satisfy parent by any further potentially satisfied follower.
