Pathway example

A following structure defines a parser for the list of steps.

Each step is determined by a turn to the certain course (left or right) or the distance to move specified using an arbitrary unit (metres or yards).

<patterns>
  <pattern>
    <node_attr attr="steps" iterator="yes">
      <one>
        <attr valuePattern="left|right">turn</attr>
        <node_attr attr="walk">
          <one>
            <attr valuePattern="\d*">metres</attr>
            <attr valuePattern="\d*">yards</attr>
          </one>
        </node_attr>
      </one>
    </node_attr>
  </pattern>
</patterns>

There is a list of input arrays in the first row of each set and the output array obtained by MapFilter_Pattern_ResultInterface::getResults() in the second one.
