FAQ

MapFilter

MapFilter

What is MapFilter Pattern?

Pattern is such class that implements MapFilter_Pattern_Interface (and optionally same other of MapFilter_Pattern interfaces). All filtering is going to be performed against this Pattern.

What is MapFilter Query?

Query is a user Key-Value storage intended to filter.

What is MapFilter Result?

Result is a complex outcome of the filtering procedure provided by MapFilter::fetchResult(). This result is provider of the same set of interfaces as given filtering Pattern.

TreePattern

TreePattern is default MapFilter_Pattern_Interface implementation.

Nodes

What is All node for?

This nodes get satisfied when node specific rules are met. All node get satisfied in case all it’s followers are satisfied.

What is One node for?

This nodes get satisfied when node specific rules are met. One node follower must be satisfied. Only one One node follower can be satisfied.

What is Opt node for?

This nodes get satisfied when node specific rules are met. Opt node is satisfied regardless of it’s followers.

What is Some node for?

This nodes get satisfied when node specific rules are met. Some node needs at least one satisfied followers to be satisfied itself.

What is KeyAttr node?

It’s the only TreePattern TreePattern node that makes decisions considering attribute value. Must have exactly one satisfied follower to be satisfied itself. All it’s followers has to have forValue attribute.

Leaves

What is Attr leaf?

Attr leaf is the only leaf of TreePattern.

Attributes

What is forValue attribute?

Attribute forValue is allowed for all TreePattern node elements that are followers of KeyAttr node. Contains regular expression to match by Key-Attr value.

What is assert attribute?

assert attribute enables to set assertions in case that attribute or group of attributes fails in satisfaction process. This attribute is allowed for all pattern elements. All sat assertions can be had by getAsserts() from parsed filter.

What is flag attribute?

flag is an universal pattern element attribute to provide some additional information from parsing process. When pattern node got satisfied it’s flag will be sat. This flags are accessible by getFlags().

What is valuePattern attribute?

valuePattern in an optional attribute of Attr leaf. Defines REGEXP to match for future attribute value. Attr leaf with this pattern can be satisfied only if both attribute is present and it’s value matches pattern.

What is default attribute?

default attribute is an optional Attr node attribute. If there is an unsatisfied Attr node with defined default attribute, default attribute will be used and node will be considered satisfied.
