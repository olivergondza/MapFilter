Usage

MapFilter

To use MapFilter there is need to define the Pattern (MapFilter::setPattern ();) describing all possible data relations and let MapFilter do all the work related with input filtering. Filtering outputs can be had by fetchResult (). This object implements the same set of interfaces as original Pattern did.

MapFilter Pattern

MapFilter Pattern is an arbitrary implementation of MapFilter_Pattern_Interface. This pattern is used during the parsing procedure to do all the work needed to obtain all result types specified by implemented interfaces. There is the default implementation of Pattern included in MapFilter package called TreePattern (MapFilter_TreePattern).
