MapFilter

MapFilter

MapFilter is extensible tool to filter key-value data structures such as associative arrays (maps or hash tables) in PHP5.

MapFilter package is distributed with default Pattern implementation. MapFilter_TreePattern enables to enforce relations between groups of data Keys, Keys and allowed Values or Values and Keys from simple Key or Value -based whitelisting to more complex structures.
