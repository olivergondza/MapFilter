Filter example

Structure of filter utility.

-h and -v are completely optional options.

-c reverses column order -l reverses line order -a reverse character order in whole file.

when -c and -l can occur simultaneously -a is completely incompatible with other types of modifications.

<pattern>
  <one>

    <attr>-h</attr>
    <attr>-v</attr>

    <some>
      <attr>-c</attr>
      <attr>-l</attr>
    </some>

    <attr>-a</attr>

  </one>
</pattern>

There is a list of input arrays in the first row of each set and the output array obtained by MapFilter_Pattern_ResultInterface::getResults() in the second one.

  public static function provideParseFilterUtility () {
  
    return Array (
        Array (
            Array (),
            Array ()
        ),
        // Valid input queries
        Array (
            Array ( '-h' => NULL ),
            Array ( '-h' => NULL )
        ),
        Array (
            Array ( '-v' => NULL ),
            Array ( '-v' => NULL )
        ),
        
        Array (
            Array ( '-a' => NULL ),
            Array ( '-a' => NULL )
        ),
        Array (
            Array ( '-c' => NULL ),
            Array ( '-c' => NULL )
        ),
        Array (
            Array ( '-l' => NULL ),
            Array ( '-l' => NULL )
        ),
        Array (
            Array ( '-c' => NULL, '-l' => NULL ),
            Array ( '-c' => NULL, '-l' => NULL )
        ),
        // Truncate invalid option
        Array (
            Array ( '-m' => NULL ),
            Array ()
        ),
        // Truncate redundant argument
        Array (
            Array ( '-h' => NULL, '-v' => NULL ),
            Array ( '-h' => NULL )
        ),
        // -a is truncated since -c and -l is defined earlier
        Array (
            Array ( '-c' => NULL, '-l' => NULL, '-a' => NULL ),
            Array ( '-c' => NULL, '-l' => NULL )
        ),
        // -c and -l are truncated since -h is defined earlier
        Array (
            Array ( '-c' => NULL, '-l' => NULL, '-h' => NULL ),
            Array ( '-h' => NULL )
        ),
        // -a is truncated since -v is defined earlier
        Array (
            Array ( '-a' => NULL, '-v' => NULL ),
            Array ( '-v' => NULL )
        ),
    );
  }
